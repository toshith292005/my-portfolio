# Elite Personal Portfolio | Venkat Sai Toshith Aleti

A production-grade, elite-level developer portfolio built for **Venkat Sai Toshith Aleti**. This project is designed to position a systems-focused, startup-ready engineer with high-end aesthetics and technical depth.

## 🚀 Features

### High-End UI/UX
- **Futuristic Aesthetic**: Developer terminal theme with neon emerald accents.
- **System Boot Loader**: Custom animated loading screen for a premium first impression.
- **Glassmorphism**: Modern glass-based design system using Tailwind CSS v4.
- **Micro-interactions**: Smooth animations and transitions powered by Framer Motion.

### Technical Depth
- **System Design View**: Interactive architecture deep-dives for every project.
- **AI Chatbot**: A terminal-style assistant trained on professional experience and skills.
- **GitHub Integration**: Live repository statistics (stars, forks) via the GitHub API.
- **Interactive Stats**: Radar charts for visualizing technical proficiency.

### Content & Management
- **Tech Blog**: A dedicated section for system design and engineering posts.
- **Admin Analytics**: A private dashboard to monitor visitor traffic and system performance.
- **SEO Optimized**: Advanced metadata, Open Graph support, and search-engine friendly structure.

## 🛠 Tech Stack

- **Frontend**: Next.js 16 (App Router), React 19
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Charts**: Chart.js, React-Chartjs-2
- **Icons**: Lucide React + Custom SVG Brand Icons
- **Deployment**: Optimized for Vercel

## 📂 Architecture

The project follows a modular, component-based architecture:

```text
/src
  /app        # App Router pages and layouts
  /components # Reusable UI and section components
  /lib        # Utility functions and constants
  /styles     # Global CSS and Tailwind v4 config
```

## 📦 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```
